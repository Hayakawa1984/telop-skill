# テロップスキル セットアップ手順

Claude Codeに「テロップつけて」と言うだけで動画に字幕がつくようになります。  
**所要時間：約15〜20分（初回のみ）**

---

## はじめる前に確認

- [ ] Macを使っている
- [ ] Claude Codeがインストールされている

---

## ステップ1：ターミナルを開く

「ターミナル」はMacに最初から入っているアプリです。

1. キーボードで **command ⌘ + スペース** を押す
2. 「ターミナル」と入力してEnterを押す
3. 黒い画面が開けばOK

---

## ステップ2：必要なツールをインストール

ターミナルに以下を**コピー＆ペースト**してEnterを押す。

```
pip3 install openai-whisper imageio-ffmpeg
```

> ⏳ 文字がたくさん流れますが、止まるまで待ってください（1〜2分）。  
> 最後に `Successfully installed` と表示されればOK。

---

## ステップ3：スキルフォルダを作る

続けてターミナルに以下を**コピー＆ペースト**してEnterを押す。

```
mkdir -p ~/.claude/skills
```

何も表示されなければ成功です。

---

## ステップ4：スキルをダウンロード＆インストール

以下を**1行ずつ**コピー＆ペーストしてEnterを押す。

**① ダウンロード**
```
cd ~/Desktop && git clone https://github.com/Hayakawa1984/telop-skill.git
```

> デスクトップに「telop-skill」フォルダが作られます。

**② インストール**
```
cp -r ~/Desktop/telop-skill/telop ~/.claude/skills/telop
```

> エラーが出なければ成功です。

---

## ステップ5：Claude Codeを再起動

Claude Codeを一度完全に閉じて、もう一度開いてください。

---

## 完了！使い方

Claude Codeを開いて、動画ファイルをドラッグ＆ドロップしてから

```
テロップつけて
```

と入力するだけです。

---

## うまくいかないとき

**「pip3: command not found」と表示された場合**  
→ Pythonが入っていません。https://www.python.org/downloads/ からインストールしてください。

**「git: command not found」と表示された場合**  
→ ターミナルで以下を実行してください：
```
xcode-select --install
```
ポップアップが出たら「インストール」を押す。完了後、ステップ4からやり直し。

**それでもわからない場合**  
→ スクリーンショットを撮って共有してください！
